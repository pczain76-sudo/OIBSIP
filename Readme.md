Oasis Infobyte – Level 2 Internship Projects

This repository contains the **Level 2 Internship Projects** completed during my internship at **Oasis Infobyte**. All projects were built using **React.js**, **Tailwind CSS**, and modern JavaScript concepts to strengthen my front-end development skills.

🚀 Completed Projects

### 1. Calculator App

* Performed basic arithmetic operations
* Responsive user interface
* Clean and modern design
* Built using React state management

### 2. Todo List Application

* Add, edit, delete, and complete tasks
* Pending and completed task management
* Local Storage integration for data persistence
* Timestamp support for tasks
* Responsive UI with Tailwind CSS

### 3. Simple Authentication System

* User registration and login
* Form validation
* Duplicate user detection
* Protected dashboard
* Session management using Local Storage
* Logout functionality
* React Router based navigation

### 4. Tribute Page

* Responsive tribute page layout
* Structured content sections
* Modern UI using Tailwind CSS
* Clean and reusable React components
